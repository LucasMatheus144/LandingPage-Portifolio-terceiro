E eu sou

Formado em Ciência da Computação, atuo como Analista de Sistemas Pleno na Interfocus Tecnologia, onde concentro meus esforços no desenvolvimento de software, com destaque para minhas competências em C#, PostgreSQL. Tenho experiência em trabalho colaborativo com equipes multidisciplinares, alinhando minhas habilidades técnicas às metas organizacionais e contribuindo para o crescimento e sucesso dos projetos. Sou motivado pela busca constante por aprimorar meus conhecimentos e desenvolver novas competências, mantendo-me atualizado com as melhores práticas do setor. Meu objetivo é promover inovações que agreguem valor, otimizem processos e impulsionem resultados de excelência para a equipe e a organização.

Habilidades

C# DotNet

Possuo conhecimento em desenvolvimento com C# e .NET, com maior domínio no uso do Entity Framework Core para manipulação de dados e construção de aplicações. PostgreSQL

Tenho conhecimento em PostgreSQL, com habilidade para realizar consultas complexas, criar e gerenciar triggers, realizar análise de dados e interpretar funções de forma eficiente, contribuindo para a otimização e o desempenho do banco de dados.

Html Css Js

Possuo conhecimento em HTML, CSS e JavaScript, permitindo criar interfaces de usuário responsivas e funcionais. Sou capaz de trabalhar com estilos personalizados, utilizando CSS para criar designs consistentes e atrativos, além de aplicar JavaScript para adicionar interatividade às páginas.

React

Tenho conhecimento no uso do React para desenvolvimento de aplicações front-end modernas e dinâmicas, utilizando seus conceitos principais como componentização, hooks e gerenciamento de estado. Também trabalho com Vite como ferramenta de build, aproveitando sua alta performance e simplicidade para configurar projetos de forma ágil. Minha experiência inclui a criação de interfaces responsivas e interativas, integrando o React com outras bibliotecas e APIs para entregar soluções eficientes e escaláveis.

Projetos

Espaço Potencial

O Espaço Potencial é um sistema desenvolvido para gestão e controle de usuários, funcionários e responsáveis, oferecendo funcionalidades específicas para organização e administração eficiente de dados. O projeto conta com uma área administrativa para realizar os cadastros das informações principais, como dados dos usuários e gerenciamento de permissões. Além disso, possui uma área financeira onde é realizada toda a movimentação financeira, incluindo a criação de contas a receber com base nas parcelas estipuladas. Foi desenvolvido utilizando .NET MVC 6, com banco de dados PostgreSQL, autenticação e autorização por meio do Identity Framework e interface construída com HTML, CSS e JavaScript

Web Scraping com Python

Durante minha estadia na faculdade, desenvolvi um bot de Web Scraping utilizando Python, projetado para analisar todo o catálogo da OLX. O bot tem como objetivo identificar o produto desejado com o menor preço no mercado, fornecendo uma análise eficiente e automatizada para facilitar a tomada de decisão na compra. A implementação incluiu técnicas de extração de dados, tratamento e comparação das informações coletadas, garantindo precisão e eficiência nos resultados.
